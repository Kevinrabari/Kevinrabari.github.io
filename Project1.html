Calculator

Calculation got easier!

About Project

Approximately, there are 2.5 billion users who have been using Android. Thus, Android is a great platform for this application to reach maximum people. So, it has great reachability. Mathematics is quite complicated for many of us and therefore, this Calculator application could save lots of time by easing calculation. So, it is problem solving as well. Users can perform calculations like division, multiplication, addition, subtraction, Percentages and by using the equal button they can get accurate results.

My Role

The Calculator is the first application that I have built using Android studio. I have implemented the design and programming of the calculator application. There are individual functions and classes are used to perform multiple tasks.
