About Project

It is a simple multiplayer game that allows users X and O to play. It has in total 4 screens. “Story by X” and “New game”/”Past game” are the main screens. The story by X : The top right corner has a “+” sign which is used to navigate to start a new game. It shows information about a happy and sad image as to whether X is won or lost and the date and time when the game was played. Also, by tapping on that it will show automation of how the game was played. New Game: On the top left corner, there will be a back button that allows users to go back to the “story by X” page. In the middle top, it will show the game state which means it is X's turn or O's turn. The players can enjoy a game and at last, it will show results as “ O won”/”X won” or “ Draw”.
