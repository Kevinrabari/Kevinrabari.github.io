Numbers Game

About Project

For iPhones, they have their OS which is known as iOS and there are over billion iPhone users. It is a simple application that has a 9-digit number (200507104), one label and a reset button. The main function of the application is that as the user tap on a number it will match the last number. For example, here the last number is 4. On the number 2, the user taps that will increase the number and stop it at number 4. Whereas, on number 7, the user taps that will decrease the number and stop it at number 4. Label shows the name of the application creator. And reset button will reset every number and will show the original number (200507104).

My Role

The numbers game is the first iOS application that I have build. I have used buttons, a grid, and a label to archive this functionality. Moreover, using shift language programming and designing has been done.
